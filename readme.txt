The screen shots are written in French. 
I employed Xamarin to develop this application. 
This solution's objective is to assist users in registering new students, controlling their attendance during sessions, saving payment receipts, and calculating net worth. 
In order to grasp the work better, directors can visualize monthly and daily income and expenses through graphs.
A database created using Laravel connects instructors and administrators. 
Depending on their role (teacher-director-secretary), each user has their own set of authorities.
